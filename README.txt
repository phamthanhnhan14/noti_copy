Getting Started
---------------
- Active virtualen
- cd <directory containing this file>

- $venv/bin/python setup.py develop # to seting up anythings
    Remind yum install python-devel mysql-devel
- $venv/bin/init_ntf_demo_db development.ini # to create new database

- $venv/bin/pserve development.ini # run server
